General Syntax

Function Declarations

Example of a multi-line function: 

    void some_function(int param1, float param2) {
        /* do something */
    }

Example of a single-line function: 

    void some_function(int param1, float param2) = /* do something */

"void" here is the return type of the function (the type of the variable that the function outputs). "int" and "float" are types of the variables you pass to the function as parameters. The built-in types are going to be explained later in this documentation. The texts within /* and */ are comments.

Example of an extension function: 

            int int.multiply_by_ten() {
                return this * 10;
            }

The first "int" here is the return type of the function, the second "int" is the type that this function extends.

Function calls

            println("Hello, World!") /* normal function call with parentheses */
            println "Hello, World!" /* normal function call without parentheses */
            5.multiply_by_ten() /* extension function call with parentheses */
            5.multiply_by_ten /* extension function call without parentheses */

Variable Declarations

Example of a variable declaration: 

    int i = 0

"int" here is the type of the variable, "i" is the name, "0" is the initial value.

Strings and characters

Single quotes are used to declare character literals, double quotes are used to declare string literals. A few things about strings: "Hello, World!" will default to a proper string when a valid String struct is available in the project, in all other cases, it'll be a zero-terminated string. Z"Hello, World!" will always be a zero-terminated string. A"Hello, World!" will always be a character array.

Arrays and pointers

Arrays and pointers are basically the same in QUPL. Here is how you can declare and access them: 

    int[]  int_pointer  = 420 /* you can think of this as an array of length one
                                  with "1" as its only element                    */
    char[] char_pointer = []0xB8000 /* pointer to the address 0x8b000 (video memory
                                        address provided by BIOS when in text mode)  */
    int[]  c_array      = int[] C(1, 2, 3, 4, 5) /* C-like array         */
    println(int_pointer[0]) /* prints 420 */
    println(c_array[0])     /* prints 1   */
    char_pointer[0] = 'H'
    char_pointer[1] = 'e'
    char_pointer[2] = 'l'
    char_pointer[3] = 'l'
    char_pointer[4] = 'o'
    char_pointer[5] = '!'
    /* dynamic arrays will be discussed later */
