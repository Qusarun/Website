Dynamic Arrays

A dynamic array can change in size, add new elements, and remove existing elements. It has a very similar declaration to the C-like arrays. 

    int[] array = int[] (1, 2, 3)
    println(array) /* 1, 2, 3 */
    array.add(4) /* add 4 to the array */
    array.add(5) /* add 5 to the array */
    println(array) /* 1, 2, 3, 4, 5 */
    array.remove(4) /* remove the 5th element */
    array.remove(3) /* remove the 4rd element */
    println(array) /* 1, 2, 3 */
